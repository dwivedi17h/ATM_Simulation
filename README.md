# ATM Simulation Console Application (Java)

This is a simple ATM simulation system written in Java. It provides core banking features like user authentication, balance check, deposit, withdrawal, money transfer, and transaction history â€” all via the command line.

## Features

- Secure login using User ID, Account Number, and PIN
- Deposit and withdraw money
- Transfer funds between users
- View transaction history
- Display current account balance
- Predefined user data stored in-memory

## Technologies Used

- Java
- Object-Oriented Programming (OOP)
- Java Collections (HashMap, ArrayList)
- Scanner for console input
